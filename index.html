Moviebackstage

电影后台管理系统

View on GitHub Download .zip Download .tar.gz

功能需求：实现电影后台管理系统功能，总体需求模块如下：

* 用户管理：用户的登录和退出。
* 新闻管理：新闻信息的管理，并提供浏览功能。
* 影片分类：对影片类别进行管理。
* 影片上传：上传新的影片。
* 影片管理：对上传影片的详细信息进行管理。

性能需求：

* 应用程序要提供友好的图形用户界面。
* 应用程序具备健壮性、稳定性和好的移植性。
* 页面要为用户提供方便的导航。

实现MVC分离

用JSP实现视图层设计，SERVLET实现控制层设计，使用DTO(JAVABEAN)、CONNETOR对象实现MODEL层设计

数据库设计要求符合第一、第二范式要求。
